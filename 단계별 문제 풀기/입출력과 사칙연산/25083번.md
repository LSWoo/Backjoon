### 문제 : 아래 예제와 같이 새싹을 출력하시오.
```Text
         ,r'"7
r`-_   ,'  ,/
 \. ". L_r'
   `~\/
      |
      |      
```

```C++
#include <iostream>

using namespace std;

int main()
{
    cout << "         ,r'\"7"<<endl;
    cout << "r`-_   ,'  ,/"<<endl;
    cout << " \\. \". L_r'"<<endl;
    cout << "   `~\\/"<<endl;
    cout << "      |"<<endl;
    cout << "      |";
}
```
